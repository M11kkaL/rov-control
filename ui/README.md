# UI

React + TypeScript + Vite frontend with an integrated WebGL/Three.js ROV simulator.

See the [project README](../README.md) for architecture, setup, and usage instructions.

## Quick start

```bash
npm install
npm run dev
```

Requires the Go backend running on `http://localhost:8080` — see [README](../README.md#quick-start).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server (http://localhost:5173) |
| `npm run build` | Type-check and production build |
| `npm run lint` | Run oxlint |
| `npm run preview` | Preview production build |
